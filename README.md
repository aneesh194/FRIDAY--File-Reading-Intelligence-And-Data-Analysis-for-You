# FRIDAY--File-Reading-Intelligence-And-Data-Analysis-for-You
FRIDAY AI is a fully offline, privacy-first AI assistant inspired by Tony Stark's FRIDAY. Built with Python, DeepSeek-R1, llama.cpp, ChromaDB, and PaddleOCR, it offers conversational AI, document analysis, OCR, code intelligence, vector memory, report generation, and an animated terminal companion—all without cloud APIs.
